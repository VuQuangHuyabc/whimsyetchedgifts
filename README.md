## 🚀 Getting Started

1. **Open the Website:**
   - Double-click `index.html` to open in your browser
   - Or serve it with a local server for best results

2. **Browse Products:**
   - Navigate to the Products section
   - Use filter buttons to browse categories
   - Click "Quick View" for product details
   - Add items to cart

3. **Manage Cart:**
   - Click the cart icon in navigation
   - Adjust quantities or remove items
   - View order summary with tax calculation

## 🎨 Customization

### Colors
Edit the CSS variables in `css/style.css`:
```css
:root {
    --primary-color: #6c63ff;
    --secondary-color: #ff6b6b;
    --accent-color: #4ecdc4;
    /* ... */
}
```

### Products
Update product data in `js/script.js`:
```javascript
const products = [
    {
        id: 1,
        name: "Product Name",
        price: 12.99,
        image: "SP1/main_images/main-image-1.jpeg",
        category: "classic",
        description: "Product description"
    }
    // Add more products...
];
```
